# 🎵 Music Set Player

单文件网页音乐播放器 — 本地音频集 + 自定义封面。

## 功能

- 📁 拖放或选择本地音频文件，自动组成音乐集播放
- 🖼️ 点击或拖放自定义封面图片
- ▶️ 播放/暂停 + 可拖拽进度条
- 🔊 音量控制
- 🔄 列表循环模式
- 💾 音量与循环设置自动记忆（LocalStorage）
- 🎨 暗色紫调主题

## 使用

双击 `music-player.html` 即可在浏览器中打开，无需服务器、无需安装。

## 技术栈

纯原生 HTML/CSS/JS，零依赖。

## 浏览器兼容

Chrome 90+ / Firefox 90+ / Edge 90+ / Safari 15+
